When the `isDisjointWith` method is called with argument _iterable_, the following steps are taken:

1. Let _set_ be the *this* value. 1. If Type(_set_) is not Object, throw a *TypeError* exception. 1. Let _hasCheck_ be ? Get(_set_, "has"). 1. If IsCallable(_hasCheck_) is *false*, throw a *TypeError* exception. 1. Let _iter_ be ? GetIterator(_iterable_). 1. Repeat, 1. Let _next_ be ? IteratorStep(_iter_). 1. If _next_ is *false*, return *true*. 1. Let _nextValue_ be ? IteratorValue(_next_). 1. Let _has_ be Call(_hasCheck_, _set_, « _nextValue_ »). 1. If _has_ is an abrupt completion, return ? IteratorClose(_iter_, _has_). 1. If _has_.[[Value]] is *true*, return *false*.
