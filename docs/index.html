    2Set.prototype.union(iterable)

    When the union method is called, the following steps are taken:

    1. Let set be the this value.
    2. If Type(set) is not Object, throw a TypeError exception.
    3. Let Ctr be ? SpeciesConstructor(set, %Set%)
    4. Let newSet be ? Construct(Ctr, set)
    5. Let adder be ? Get(newSet, "add").
    6. If IsCallable(adder) is false, throw a TypeError exception.
    7. Let iter be GetIterator(iterable).
    8. If iter is undefined, throw a TypeError exception.
    9. Repeat,
      1. Let next be IteratorStep(iter).
      2. If next is false, return newSet.
      3. Let nextValue be ? IteratorValue(next).
      4. Let status be Call(adder, newSet, « nextValue.[[Value]] »)
      5. If status is an abrupt completion, return ? IteratorClose(iter, status)

    The length property of the union method is 1.

    3Set.prototype.intersect(iterable)

    When the intersect method is called, the following steps are taken:

    1. Let set be the this value.
    2. If Type(set) is not Object, throw a TypeError exception.
    3. Let Ctr be ? SpeciesConstructor(set, %Set%).
    4. Let newSet be ? Construct(Ctr).
    5. Let hasCheck be ? Get(set, "has").
    6. If IsCallable(hasCheck) is false, throw a TypeError exception.
    7. Let adder be ? Get(newSet, "add").
    8. If IsCallable(adder) is false, throw a TypeError exception.
    9. Let iter be GetIterator(iterable).
    10. If iter is undefined, throw a TypeError exception.
    11. Repeat,
      1. Let next be IteratorStep(iter).
      2. If next is false, return newSet.
      3. Let nextValue be ? IteratorValue(next).
      4. Let has be Call(hasCheck, set, « nextValue.[[Value]] »)
      5. If has is an abrupt completion, return ? IteratorClose(iter, has)
      6. If has is true,
        1. Let status be Call(adder, newSet, « nextValue.[[Value]] »)
        2. If status is an abrupt completion, return ? IteratorClose(iter, status)

    The length property of the intersect method is 1.

    6Set.prototype.isSubset(iterable)

    When the isSubset method is called, the following steps are taken:

    1. Let set be the this value.
    2. If Type(set) is not Object, throw a TypeError exception.
    3. If set does not have a [[SetData]] internal slot, throw a TypeError exception.
    4. Let entries be the List that is set.[[SetData]].
    5. If Type(iterable) is not Object, throw a TypeError exception.
    6. If iterable does not have [[SetData]] internal slot,
      1. Let otherSet be Construct(%Set%, iterable).
    7. Else, let otherSet be iterable.
    8. Let hasCheck be ? Get(otherSet, "has").
    9. If IsCallable(hasCheck) is false, throw a TypeError exception.
    10. For each e that is an element of entries, do
      1. Let has be Call(hasCheck, otherSet, « e »)
      2. If has is false, return false.
    11. return true.

    The length property of the isSubset method is 1.

    7Set.prototype.isSuperset(iterable)

    When the isSuperset method is called, the following steps are taken:

    1. Let set be the this value.
    2. If Type(set) is not Object, throw a TypeError exception.
    3. If set does not have a [[SetData]] internal slot, throw a TypeError exception.
    4. Let hasCheck be ? Get(set, "has").
    5. If IsCallable(hasCheck) is false, throw a TypeError exception.
    6. Let iter be GetIterator(iterable).
    7. If iter is undefined, throw a TypeError exception.
    8. Repeat,
      1. Let next be IteratorStep(iter).
      2. If next is false, return true.
      3. Let nextValue be ? IteratorValue(next).
      4. Let has be Call(hasCheck, set, « e »).
      5. If has is an abrupt completion, return ? IteratorClose(iter, has).
      6. If has is false, return false.

    The length property of the isSuperSet method is 1.
